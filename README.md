# Bank Account System

A simple console-based **Bank Account System** developed in **C++** to demonstrate object-oriented programming concepts. The project simulates basic banking operations using different account types, allowing users to perform transactions and manage account information.

## Features

- Create bank accounts
- Support for Savings and Current accounts
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Display account details
- Menu-driven console interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Virtual Functions
- Vectors
- File Handling (Optional)

## How to Run

### Using g++

Compile the project:

```bash
g++ main.cpp -o bank
```

Run the program:

```bash
./bank
```

### Using CodeBlocks or Visual Studio

1. Create a new C++ Console Project.
2. Replace the default source file with `main.cpp`.
3. Build and run the project.
